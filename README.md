# US Dividend-ETF Income Planner

Working prototype for screening US income ETFs, predicting dividend-cut risk,
optimizing income portfolios, and visualizing the latest run in a static
dashboard.

## What It Does

1. Builds a hardcoded universe of US dividend, covered-call, bond, REIT,
   preferred, multi-asset, international-dividend, and MLP income ETFs.
2. Fetches daily close prices, monthly distributions, and basic ETF info through
   `yfinance`.
3. Engineers monthly ETF features and labels dividend cuts.
4. Trains a balanced logistic-regression classifier for 12-month dividend-cut
   risk.
5. Optimizes Safe / Balanced / High-Income portfolios with a mean-variance
   optimizer.
6. Exports the latest run to `web/public/data/latest.json`.
7. Serves a deployable static dashboard.

## Run the CLI

Install dependencies:

```bash
python3 -m pip install -r requirements.txt
```

Run with live or cached yfinance data:

```bash
python3 main.py --budget 50000 --sleep 0
```

Run with deterministic offline demo data:

```bash
python3 main.py --budget 50000 --demo-data
```

Disable the per-ETF max-weight constraint:

```bash
python3 main.py --budget 50000 --sleep 0 --no-max-weight
```

## Export UI Data

```bash
python3 export_ui_data.py --budget 50000 --sleep 0
```

This writes:

```text
web/public/data/latest.json
```


- `vercel.json`
- `package.json`
- `web/`
- `web/public/data/latest.json`

The current deployed dashboard is:

```text
https://dividend-etf-income-planner-static.vercel.app
```

## Important Files

- `universe.py`: ETF candidate universe and screening
- `data.py`: yfinance fetch, normalization, caching, demo data
- `features.py`: feature engineering and dividend-cut labels
- `model.py`: classifier, walk-forward validation, latest scoring
- `optimizer.py`: mean-variance optimizer and portfolio plans
- `main.py`: CLI pipeline
- `export_ui_data.py`: dashboard JSON exporter
- `web/`: static dashboard
- `outputs/`: saved text outputs from prior runs
